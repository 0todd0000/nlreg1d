Hypothesis tests¶

This study uses the spm1d package to conduct hypothesis testing. This package uses statistical parametric mapping (SPM) to conduct simultaneous inference across the domain, via a random field theory (RFT) correction for multiple comparisons. This RFT correction is a smoothness-based correction that accurately controls Type I error (usually at $\alpha=0.05$) for arbitrarily smooth (or rough) data when the data are normally distributed at each point in the domain. RFT theory as it pertains to 1D data is summarized in this paper, and a list of key SPM and RFT references is available here.

Let's conduct a two-sample t-test use the SimulatedB dataset.

# load data:
dataset = nl.data.SimulatedB()
y0,y1   = dataset.get_dv_by_group()
print(y0.shape)
print(y1.shape)
print()

# conduct hypothesis test
spmi    = spm1d.stats.ttest2( y1, y0 ).inference(0.05)
print(spmi)


# plot results
fig,axs = plt.subplots(1, 2, figsize=(11,4))
ax0,ax1 = axs

h0      = ax0.plot(y0.T, 'k', lw=0.5)[0]
h1      = ax0.plot(y1.T, 'c', lw=0.5)[0]
h0      = ax0.plot(y0.mean(axis=0), 'k', lw=5)[0]
h1      = ax0.plot(y1.mean(axis=0), 'c', lw=5)[0]
ax0.legend( [h0,h1], ['Group 0 mean', 'Group 1 mean'])

spmi.plot(ax=ax1)
spmi.plot_threshold_label(fontsize=12)
spmi.plot_p_values(size=14)
ax1.set_ylabel('t-value')

plt.tight_layout()
plt.show()

Notes:

  • The maximim t-value crosses the critical threshold of tc = 3.362, so the null hypothesis of equivalent group means is rejected at $\alpha=0.05$
  • The critical threshold ( tc = 3.362 ) can be interpreted as follows: Gaussian random fields with the identical smoothness to the observed residuals will yield a maximum t-value that traverses tc with a probability of $\alpha=0.05$
  • The probability value (p-value) associated with the observed suprathreshold cluster is p = 0.002, which is interpreted in a post hoc sense as follows: Gaussian random fields with the identical smoothness to the observed residuals will yield a suprathreshold cluster with the observed temporal extent with a probability of $p=0.002$
